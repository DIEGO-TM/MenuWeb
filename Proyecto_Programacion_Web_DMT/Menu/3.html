Listas en HTML

Tipos de Listas

HTML ofrece tres tipos principales de listas:

  • Listas Ordenadas (<ol>)
  • Listas No Ordenadas (<ul>)
  • Listas de Definición (<dl>)

Lista Ordenada (ol)

Ejemplo de lista ordenada:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Atributos de ol:

  • type: "1", "A", "a", "I", "i"
  • start: número inicial
  • reversed: orden inverso

Lista No Ordenada (ul)

Ejemplo de lista no ordenada:

  • Elemento con viñeta
  • Otro elemento
  • Un elemento más

Lista de Definición (dl)

Ejemplo de lista de definición:

HTML
HyperText Markup Language - Lenguaje de marcado para crear páginas web
CSS
Cascading Style Sheets - Hojas de estilo en cascada

Estilizando Listas con CSS

Propiedades CSS comunes para listas:

  • list-style-type: Define el tipo de marcador
  • list-style-image: Usa una imagen como marcador
  • list-style-position: Posición del marcador
  • margin y padding: Espaciado

Ejemplo de CSS para listas:

ul {
    list-style-type: square;
    margin-left: 20px;
}

ol {
    list-style-type: upper-roman;
    padding-left: 30px;
}

dl {
    margin: 15px 0;
}

dt {
    font-weight: bold;
}

dd {
    margin-left: 30px;
}

Las listas son elementos fundamentales en HTML para organizar y estructurar contenido de manera clara y accesible.
