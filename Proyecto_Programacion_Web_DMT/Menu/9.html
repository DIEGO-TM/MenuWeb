Elementos Interactivos y Gráficos en HTML y CSS

Canvas HTML5

El elemento canvas permite dibujar gráficos mediante JavaScript:

Ejemplo básico de Canvas:

<canvas id="miCanvas" width="300" height="200"></canvas>

<script>
const canvas = document.getElementById('miCanvas');
const ctx = canvas.getContext('2d');
ctx.fillStyle = 'blue';
ctx.fillRect(10, 10, 100, 100);
</script>

SVG (Scalable Vector Graphics)

SVG permite crear gráficos vectoriales escalables:

Ejemplo de SVG:

<svg width="200" height="200">
    <circle cx="100" cy="100" r="50" fill="red"/>
    <rect x="50" y="50" width="100" height="100" 
          fill="blue" opacity="0.5"/>
</svg>

Elementos Interactivos

HTML5 incluye elementos nativamente interactivos:

  • <details> y <summary>: Para contenido expandible
  • <dialog>: Para ventanas modales
  • <progress>: Para barras de progreso
  • <meter>: Para medidores

Ejemplos:

<details>
    <summary>Ver más información</summary>
    <p>Contenido expandible aquí...</p>
</details>

<progress value="70" max="100">70%</progress>

<meter value="0.6">60%</meter>

Efectos Interactivos con CSS

Ejemplos de efectos con CSS:

.elemento-hover {
    transition: all 0.3s ease;
}

.elemento-hover:hover {
    transform: translateY(-5px);
    box-shadow: 0 5px 15px rgba(0,0,0,0.3);
}

.elemento-activo:active {
    transform: scale(0.95);
}

@keyframes flotante {
    0% { transform: translateY(0); }
    50% { transform: translateY(-10px); }
    100% { transform: translateY(0); }
}

Los elementos interactivos y gráficos mejoran significativamente la experiencia del usuario y permiten crear interfaces más dinámicas y atractivas.
