Estructura del Documento HTML

¿Qué es la estructura de un documento HTML?

La estructura básica de un documento HTML consta de las siguientes etiquetas principales:

  • <!DOCTYPE html>: Define el tipo de documento y asegura que el navegador interprete el código HTML5 correctamente.
  • <html>: Es el contenedor principal que engloba todo el documento HTML. El atributo lang especifica el idioma del contenido.
  • <head>: Contiene metadatos y enlaces a recursos externos como:
    • Hojas de estilo CSS
    • Scripts JavaScript
    • Fuentes web
    • Favicon
    • Meta tags para SEO
  • <body>: Contiene todo el contenido visible de la página como:
    • Textos y párrafos
    • Imágenes y multimedia
    • Enlaces y botones
    • Formularios
    • Elementos estructurales (header, nav, main, footer)

Ejemplo de Estructura Básica

Estructura HTML completa:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi primera página</title>
</head>
<body>
    <h1>¡Hola Mundo!</h1>
    <p>Este es un ejemplo de la estructura básica de un documento HTML.</p>
</body>
</html>
