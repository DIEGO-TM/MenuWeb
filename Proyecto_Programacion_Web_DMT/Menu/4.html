Enlaces y Navegación en HTML

Enlaces HTML Básicos

La etiqueta básica para crear enlaces es <a>:

  • href: URL o ruta del destino
  • target: Dónde abrir el enlace (_blank, _self, etc.)
  • title: Texto descriptivo al pasar el mouse

Ejemplo:

<a href="https://www.ejemplo.com" target="_blank">
    Visitar sitio web
</a>

Tipos de Enlaces

  • Enlaces absolutos: URLs completas 
    <a href="https://www.ejemplo.com">Sitio web</a>
  • Enlaces relativos: Rutas dentro del sitio 
    <a href="paginas/contacto.html">Contacto</a>
  • Enlaces de correo: Para abrir cliente de correo 
    <a href="mailto:correo@ejemplo.com">Enviar email</a>
  • Enlaces de teléfono: Para dispositivos móviles 
    <a href="tel:+123456789">Llamar</a>
  • Enlaces internos: A secciones de la misma página 
    <a href="#seccion">Ir a sección</a>

Estilizando Enlaces con CSS

Estados de los enlaces:

/* Enlace no visitado */
a:link {
    color: blue;
    text-decoration: none;
}

/* Enlace visitado */
a:visited {
    color: purple;
}

/* Al pasar el mouse */
a:hover {
    color: red;
    text-decoration: underline;
}

/* Al hacer clic */
a:active {
    color: orange;
}

Navegación Estructurada

Elementos semánticos para navegación:

  • <nav>: Sección de navegación principal
  • <menu>: Lista de comandos o enlaces
  • <breadcrumb>: Navegación de migas de pan

Ejemplo de menú de navegación:

<nav>
    <ul>
        <li><a href="inicio.html">Inicio</a></li>
        <li><a href="productos.html">Productos</a></li>
        <li><a href="contacto.html">Contacto</a></li>
    </ul>
</nav>

Los enlaces son fundamentales para la navegación web y la interconexión de contenidos. Un buen diseño de navegación mejora la experiencia del usuario.
