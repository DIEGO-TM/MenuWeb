Formularios y Entrada de Datos

Elementos Básicos de Formularios

Los formularios HTML permiten recopilar datos del usuario. Aquí los elementos más comunes:

  • <form>: Contenedor del formulario
  • <input>: Campo de entrada
  • <textarea>: Área de texto multilínea
  • <select>: Lista desplegable
  • <button>: Botón

Ejemplo de Formulario

<form action="/procesar" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" required>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">

    <label for="mensaje">Mensaje:</label>
    <textarea id="mensaje" name="mensaje"></textarea>

    <button type="submit">Enviar</button>
</form>

Tipos de Input

El elemento input tiene varios tipos:

  • text: Texto simple
  • password: Contraseña
  • email: Correo electrónico
  • number: Números
  • date: Fecha
  • checkbox: Casillas de verificación
  • radio: Botones de opción
  • file: Subida de archivos

Estilizando Formularios con CSS

Estilos básicos para formularios:

form {
    max-width: 500px;
    margin: 20px auto;
    padding: 20px;
}

input, textarea, select {
    width: 100%;
    padding: 8px;
    margin: 8px 0;
    border: 1px solid #ddd;
    border-radius: 4px;
    box-sizing: border-box;
}

label {
    display: block;
    margin: 10px 0 5px;
    font-weight: bold;
}

button {
    background-color: #4CAF50;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 4px;
    cursor: pointer;
}

button:hover {
    background-color: #45a049;
}

input:focus, textarea:focus {
    border-color: #2196F3;
    outline: none;
    box-shadow: 0 0 5px rgba(33,150,243,0.3);
}

Validación de Formularios

HTML5 proporciona atributos para validación:

  • required: Campo obligatorio
  • pattern: Patrón regex
  • minlength/maxlength: Longitud del texto
  • min/max: Valores numéricos

Los formularios son esenciales para la interacción con el usuario. Un buen diseño y validación mejoran la experiencia del usuario y la calidad de los datos recopilados.
