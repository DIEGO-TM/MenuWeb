Etiquetas Obsoletas en HTML5

Elementos y atributos que ya no se recomiendan usar en HTML5

Etiquetas HTML Obsoletas

Elementos que no deben usarse:

<!-- No usar estas etiquetas -->
<center>Texto centrado</center>
<font face="Arial" size="3">Texto con fuente</font>
<basefont>
<big>Texto grande</big>
<strike>Texto tachado</strike>
<tt>Texto monoespaciado</tt>
<frame>
<frameset>
<noframes>

Alternativas Modernas

Soluciones con CSS:

/* En lugar de <center> */
.texto-centrado {
    text-align: center;
}

/* En lugar de <font> */
.texto-personalizado {
    font-family: Arial, sans-serif;
    font-size: 16px;
}

/* En lugar de <big> */
.texto-grande {
    font-size: 1.2em;
}

/* En lugar de <strike> */
.texto-tachado {
    text-decoration: line-through;
}

Atributos Obsoletos

  • align (usar CSS text-align o flexbox)
  • bgcolor (usar CSS background-color)
  • border en imágenes (usar CSS border)
  • cellpadding/cellspacing en tablas (usar CSS padding)
  • width/height como atributos (usar CSS)

Buenas Prácticas

  • Usar CSS para estilos y diseño
  • Mantener la semántica HTML5
  • Separar contenido de presentación
  • Utilizar elementos semánticos modernos
  • Validar el código HTML regularmente

Es importante mantenerse actualizado con los estándares web modernos y evitar el uso de elementos obsoletos para garantizar la compatibilidad y el mantenimiento del código.
