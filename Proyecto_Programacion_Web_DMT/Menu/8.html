Elementos Semánticos y de Sección en HTML

Elementos Semánticos Principales

Los elementos semánticos ayudan a dar significado y estructura al contenido:

  • <header>: Encabezado de la página o sección
  • <nav>: Navegación principal
  • <main>: Contenido principal
  • <article>: Contenido independiente y autocontenido
  • <section>: Sección temática del contenido
  • <aside>: Contenido relacionado indirectamente
  • <footer>: Pie de página o sección

Estructura Semántica Básica

Ejemplo de estructura:

<body>
    <header>
        <nav>
            <!-- Navegación -->
        </nav>
    </header>

    <main>
        <article>
            <section>
                <!-- Contenido -->
            </section>
        </article>
        
        <aside>
            <!-- Contenido relacionado -->
        </aside>
    </main>

    <footer>
        <!-- Pie de página -->
    </footer>
</body>

Estilizando Elementos Semánticos

CSS común para elementos semánticos:

header {
    background-color: #333;
    color: white;
    padding: 1rem;
}

nav {
    background-color: #f4f4f4;
    padding: 0.5rem;
}

main {
    max-width: 1200px;
    margin: 0 auto;
    padding: 1rem;
}

article {
    background-color: white;
    padding: 1rem;
    margin-bottom: 1rem;
    border-radius: 4px;
}

section {
    margin-bottom: 2rem;
}

aside {
    background-color: #f9f9f9;
    padding: 1rem;
    border-left: 3px solid #333;
}

footer {
    background-color: #333;
    color: white;
    padding: 1rem;
    text-align: center;
}

Beneficios del HTML Semántico

  • Mejor accesibilidad para lectores de pantalla
  • Código más mantenible y legible
  • Mejor SEO (Optimización para motores de búsqueda)
  • Estructura clara y significativa
  • Facilita el desarrollo responsive

El HTML semántico es fundamental para crear sitios web modernos, accesibles y bien estructurados.
