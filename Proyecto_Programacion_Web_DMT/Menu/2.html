Etiquetas de Contenido y Texto en HTML

Etiquetas de Encabezado

HTML proporciona seis niveles de encabezados:

  • <h1> - Encabezado principal
  • <h2> - Encabezado secundario
  • <h3> hasta <h6> - Subencabezados

Etiquetas de Texto

  • <p> - Define un párrafo
  • <span> - Contenedor en línea para texto
  • <strong> - Texto en negrita con importancia semántica
  • <em> - Texto enfatizado (generalmente en cursiva)
  • <mark> - Texto resaltado
  • <sub> - Texto subíndice
  • <sup> - Texto superíndice

Ejemplos Prácticos

Texto formateado:

Este es un texto en negrita

Este es un texto enfatizado

Este es un texto resaltado

Fórmula química: H2O

Potencia: x2

Etiquetas de Formato Especial

  • <blockquote> - Para citas largas
  • <q> - Para citas cortas
  • <code> - Para código de programación
  • <pre> - Para texto preformateado
  • <br> - Salto de línea
  • <hr> - Línea horizontal

Estas son las etiquetas más comunes para el manejo de texto en HTML. Recuerda que el uso apropiado de estas etiquetas mejora la semántica y accesibilidad de tu sitio web.
