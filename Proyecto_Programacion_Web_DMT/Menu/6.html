Tablas en HTML y CSS

Estructura Básica de una Tabla

Las tablas en HTML se construyen usando las siguientes etiquetas principales:

  • <table>: Define la tabla
  • <tr>: Define una fila
  • <th>: Define una celda de encabezado
  • <td>: Define una celda de datos

Ejemplo de Tabla Básica

<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>

Elementos Adicionales de Tabla

  • <thead>: Agrupa el contenido del encabezado
  • <tbody>: Agrupa el contenido principal
  • <tfoot>: Agrupa el contenido del pie
  • <caption>: Define un título para la tabla

Atributos de Tabla

  • colspan: Combina columnas
  • rowspan: Combina filas
  • border: Define el borde (aunque se recomienda CSS)

Estilizando Tablas con CSS

Propiedades CSS comunes para tablas:

table {
    border-collapse: collapse;
    width: 100%;
    margin: 20px 0;
}

th, td {
    border: 1px solid #ddd;
    padding: 8px;
    text-align: left;
}

th {
    background-color: #f2f2f2;
    font-weight: bold;
}

tr:nth-child(even) {
    background-color: #f9f9f9;
}

tr:hover {
    background-color: #f5f5f5;
}

Tabla Responsiva

Para hacer una tabla responsiva, puedes envolverla en un contenedor con scroll horizontal:

.table-container {
    overflow-x: auto;
    max-width: 100%;
}

Las tablas son herramientas poderosas para mostrar datos tabulares. Sin embargo, no se recomienda usarlas para el diseño de la página.
