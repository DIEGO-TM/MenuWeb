Meta-información y Recursos en HTML y CSS

Meta Tags Esenciales

Meta tags básicos:

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Descripción de la página">
<meta name="keywords" content="palabras, clave, relevantes">
<meta name="author" content="Nombre del autor">

Meta Tags para SEO

Meta tags para redes sociales:

<!-- Open Graph (Facebook) -->
<meta property="og:title" content="Título">
<meta property="og:description" content="Descripción">
<meta property="og:image" content="imagen.jpg">

<!-- Twitter Cards -->
<meta name="twitter:card" content="summary">
<meta name="twitter:title" content="Título">
<meta name="twitter:description" content="Descripción">

Recursos y Enlaces

Enlaces a recursos externos:

<link rel="stylesheet" href="styles.css">
<link rel="icon" href="favicon.ico">
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preload" href="imagen.jpg" as="image">

Optimización de Recursos

CSS para optimización de recursos:

/* Lazy loading de imágenes */
img {
    loading: lazy;
}

/* Fuentes optimizadas */
@font-face {
    font-family: 'MiFuente';
    src: url('fuente.woff2') format('woff2'),
         url('fuente.woff') format('woff');
    font-display: swap;
}

Buenas Prácticas

  • Usar meta tags relevantes para SEO
  • Optimizar recursos para mejor rendimiento
  • Implementar lazy loading cuando sea posible
  • Mantener las meta descripciones concisas y relevantes
  • Utilizar favicon y recursos de marca apropiados

La meta-información y la gestión adecuada de recursos son fundamentales para el SEO y el rendimiento de un sitio web.
